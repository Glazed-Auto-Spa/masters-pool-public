    Rules & Payouts

    Main Scoring

    • Main event buy-in is $25 per participant.
    • Main event payout is winner-take-all (entire main pot to 1st place).
    • Each participant has 8 picks. All 8 picks count each day.
    • Daily total = sum of all pick scores for that day (Thu/Fri/Sat/Sun).
    • Event total = Thu + Fri + Sat + Sun (lowest total wins).
    • Missed cut, WD, or DQ: Thursday and Friday use that golfer’s live scores to par (rounds 1–2). Saturday and Sunday still count the pick: each weekend day is the floor of the average of Thursday and Friday to par (for example +5 and +7 → +6 on both Sat and Sun; +5 and +5 → +5 each). If only a combined 36-hole total exists with no per-round split, each weekend day uses half that total rounded down. The pool ignores placeholder weekend rows from the feed for these players so the average rule always applies.

    Side Games

    • Eagles: +$50 for each eagle by any player on your roster.
    • Aces: +$100 for each ace (hole-in-one) on your roster.
    • Birdie streaks: starts at 3 straight birdie-or-better holes for +$50, then +$50 for each additional consecutive birdie-or-better hole.
    • Daily winner: +$50 each day (Thu/Fri/Sat/Sun) to the single lowest team score that day.
    • Streak qualifying holes: birdie, eagle, ace. Any other result ends the streak.

    Tiebreaker & Feed Notes

    • Tiebreaker is predicted final winning score to par after Round 4.
    • If event totals tie, closest tiebreak prediction wins.
    • Last update and feed timestamps are shown in Phoenix local time (MST).
    • Live Event Feed shows latest 10 updates; use “Show 10 more” to expand.

    Main event pot is buy-in driven ($25 x participant count) and paid winner-take-all. Side bets (Eagles/Aces/Streaks/Daily Winners) are settled across all participants.

    Payout structure & settlement

    No money has moved through this app. This is an informal worksheet from the same numbers as Live Standings. Final settlement is between participants (and any host who collected buy-ins). Numbers update when the feed refreshes; use the payout integrity badge above when the tournament is complete.

    How it works: Everyone is treated as having put in the $25 main buy-in. The leader board’s main event payout is winner-take-all (full pot to 1st). Side Bet Net is the zero-sum settlement for Eagles, Aces, Streaks, and Daily winners (each dollar one person wins, others effectively fund). Your net position is main + side net − buy-in. The “who pays whom” list is a simple way to clear those positions with the fewest transfers (you may settle differently if everyone agrees).
